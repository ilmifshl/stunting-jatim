## Catatan Teknis untuk Pengolahan Data
Jika data ini akan digunakan untuk analisis runtun waktu (*time-series*) pada aplikasi web:
1. **Identifikasi Wilayah:** Gunakan kolom `Kabupaten/Kota` sebagai kunci unik untuk melakukan *join* dengan data `nama_wilayah` pada file GeoJSON.
2. **Kalkulasi Selisih (Delta):** Kamu bisa menambahkan logika di front-end untuk menghitung `Delta (%) = % 2023 - % 2022` guna memvisualisasikan wilayah mana yang mengalami percepatan sanitasi paling tinggi.
3. **Penyimpanan Database:** Disarankan menyimpan kolom persentase sebagai tipe data `float` untuk mempermudah operasi matematika di kemudian hari.
